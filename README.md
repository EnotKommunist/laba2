# Балабанов Александр БАС-1 Лабораторная 2
**Задание 1**

**Задача 1**

**Текст задачи**

Получить список из минимальных цифр натуральных чисел, содержащихся в исходном списке

**Алгоритм решения**

Программа запрашивает у пользователя количество элементов count_el. Если count_el больше нуля, программа запрашивает ввод count_el чисел и сохраняет их в список numbers. Затем вызывается функция FindMinNum, которая для каждого числа из numbers находит минимальную цифру с помощью рекурсивной функции MinNum и возвращает список этих минимальных цифр. После этого программа выводит исходный список чисел и список их минимальных цифр. Если введённое count_el меньше или равно нулю, программа выводит сообщение об ошибке. В конце программа возвращает 0.

**Задание 2**

**Задача 1**

Текст задачи

Найти количество элементов списка, в которых встречается заданная цифра.

**Алгоритм решения**

Программа запрашивает у пользователя количество элементов count_el, затем вводит count_el чисел и сохраняет их в список numbers. После этого запрашивается искомая цифра required_number. Для каждого числа из списка вызывается рекурсивная функция RecNum, которая проверяет, содержит ли число эту цифру, и возвращает 1 (если содержит) или 0 (если нет). Результаты сохраняются в список list_num. Затем с помощью List.fold суммируются все элементы list_num, что даёт количество чисел, содержащих искомую цифру. Программа выводит это число и завершается с кодом 0.
